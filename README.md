Створіть додаток Django для управління завданнями, де користувачі можуть:
- Переглядати список завдань.
- Створювати нове завдання.
- Видаляти існуюче завдання.
Вимоги:
- Реалізувати маршрути для наступних операцій:
/tasks/ – перегляд усіх завдань.
/tasks/add/ – додавання нового завдання.
/tasks/delete/id/ – видалення завдання за ідентифікатором.

Опишіть клас задачі:
- загловок задачі
- текст задачі
- дата початку
- дата завершення
